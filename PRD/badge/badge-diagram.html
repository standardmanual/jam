# 그룹명 카테고리 조건 타입 Common Rare Legendary Mythic

JAM! 배지는 같은 이름 안에서 Common → Rare → Legendary → Mythic으로 성장하는 티어 구조를 가진다. 엔진은 유저가 이미 보유한 최상위 티어보다 높은 것만 평가하며, 동시에 여러 조건을 통과해도 가장 높은 레어리티 1개만 발급한다.

성장 티어 정책
같은 이름 그룹 내에서
이미 보유한 티어보다 낮거나 같으면 건너뜀.
항상 최상위 미보유 1개만 발급.
진행 트랙 중복 제거
순수 거리/횟수 조건 배지는 같은 activity_type 내에서 최고값 1개만.
복합 조건(고도, 속도 포함)은 트랙 독립.
홍수 방지 (Flood Cap)
30일 윈도우 기준 activity_type당 최대 3개.
초과 시 mythic 우선 보존, 낮은 티어부터 탈락.
중복 발급 방지
DB INSERT 시 23505 (unique violation) 오류 발생 시 무시하고 계속.
이미 발급된 배지는 재발급 없음.
예시: "첫 숨결" 그룹 (로드러닝 누적 거리)
시나리오: 총 누적 거리 45km 달성 시
Common
≥ 3km
⊘ 건너뜀 (기 보유)
Rare
≥ 9km
⊘ 건너뜀 (기 보유)

≥ 21km ✓
⊘ 건너뜀 (더 높은 티어 있음)

≥ 42km → ✗ (45km 부족)
미달
→ Legendary 발급 (45km ≥ 21km, 42km 미달로 Mythic 제외 → 가장 높은 통과 티어)
홍수 방지 시각화 — 30일 내 road_running 발급 현황
