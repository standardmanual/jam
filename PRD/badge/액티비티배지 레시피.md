## 설계 원칙

| # | 원칙 | 내용 |
|---|------|------|
| 1 | **첫 싱크 Common 전용** | `initial_sync_done=false` 동안 Common 등급만 발급. Rare 이상 차단 |
| 2 | **크로스게이트 선행 배지** | Rare 이상은 동일 종목 다른 속성 배지(any tier) 보유 필수 (OR 조건) |
| 3 | **성장 티어 정책** | 같은 이름 그룹에서 이미 보유한 등급 이하 재발급 없음 |
| 4 | **진행 트랙 중복 제거** | 단일 조건 배지는 동일 트랙 내 최고값 1개만 발급 |
| 5 | **홍수 방지** | 30일 롤링 윈도우 / activity_type당 최대 3개 |

#### T7 — 알파인 트레일러 | 속성: 등반 × 혹한 | 조건: `elevation_gain_m` + `temperature_max_c` (복합 AND, 이력 전반 독립 평가) *(카테고리 2 — 복합속성)*

| 등급 | 조건값 | 배지 설명 |
|------|--------|----------|
| Common | 300 m & 10°C | 고도와 추위의 첫 만남. 섬데이 야생이 두 조건을 동시에 인식합니다. |
| Rare | 1,000 m & 5°C | 높고 차가운 트레일. 그루터기 살롱이 두 기록을 함께 주목합니다. |
| Legendary | 2,500 m & 0°C | 누적 고도와 영하 트레일. 블랙 트랙의 알파인 전사 명단에 이름이 오릅니다. |
| Mythic | 5,000 m & -5°C | 극한 고도와 극한 혹한. 화이트 룸은 두 알파인 코드를 해독한 자에게 열립니다. |

선행 배지 (Rare 이상): `수직의 도전` OR `혹한의 트레일러`

#### T8 — 새벽 야생인 | 속성: 새벽 빈도 | 조건: `time_range(05:00~09:00)` + `weekly_count` (복합 AND, 이력 전반 독립 평가) *(카테고리 3 — 리텐션)*

| 등급 | 조건값 | 배지 설명 |
|------|--------|----------|
| Common | 주 1회 | 새벽 5~9시 트레일 주 1회. 섬데이 야생의 여명이 당신의 발소리를 기억합니다. |
| Rare | 주 2회 | 새벽 주 2회 트레일. 그루터기 살롱이 해 뜨기 전 야생의 주자를 주목합니다. |
| Legendary | 주 3회 | 새벽 주 3회. 블랙 트랙이 여명의 트레일러를 전설로 새깁니다. |
| Mythic | 주 4회 | 새벽 주 4회 트레일. 화이트 룸은 야생의 새벽을 지배하는 자에게 열립니다. |

선행 배지 (Rare 이상): `야생의 첫발` OR `트레일 루틴`

### prerequisite_badge_names 동작

- Rare / Legendary / Mythic의 `condition_json`에 포함
- Common은 prerequisite 없음
- 엔진 Step 3 C-1에서 처리: `ownedBadgeNames` 집합과 OR 매칭
- 이름 기반: 해당 배지의 어느 등급이든 소유 시 통과

### condition_json 예시


### 카테고리 2 복합 배지 평가 주의

카테고리 2 배지 (R7, C7, H7, T7)는 두 조건이 **이력 전반에서 독립 평가**된다. 같은 세션에서 동시 충족하지 않아도 발급된다.

| 배지 | 평가 방식 |
|------|-----------|
| R7 스피드 엔듀러 | 역대 최고 평균속도 ≥ X **AND** 역대 최장 달리기 ≥ Y — 다른 세션 가능 |
| C7 산악 라이더 | 역대 최고 속도 ≥ X **AND** 누적 고도 ≥ Y — 독립 이력 |
| H7 혹한 장정 | 기록된 최저 기온 ≤ X **AND** 역대 최장 산행 ≥ Y — 독립 이력 |
| T7 알파인 트레일러 | 누적 고도 ≥ X **AND** 기록된 최저 기온 ≤ Y — 독립 이력 |

단일 세션 동시 충족을 요구하려면 엔진 수정이 필요하다.

## 진행 예시

### 로드러닝 유저

```
첫 싱크:
  → 첫 숨결 Common (3km 누적)
  → 리듬의 발견 Common (시속 7km 달성)
  → 달리기의 루틴 Common (이번 주 2번)

이후 싱크:
  → 첫 숨결 Rare (20km) ← 리듬의 발견 Common 보유 → 해금
  → 지구력의 전사 Common (20분 달리기)
  → 달리기의 루틴 Rare (주 3회) ← 첫 숨결 Common 보유 → 해금

더 이후:
  → 달리기의 연결 Common (3일 연속)
  → 달리기의 연결 Rare (7일 연속) ← 달리기의 루틴 Common 보유 → 해금
```

### 사이클 유저

```
첫 싱크:
  → 두 바퀴의 시작 Common (10km)
  → 언덕의 도전자 Common (100m 누적 고도)

이후 싱크:
  → 두 바퀴의 시작 Rare (50km) ← 언덕의 도전자 Common 보유 → 해금
  → 장거리 항속 Common (60분 라이딩)
  → 언덕의 도전자 Rare (500m) ← 두 바퀴의 시작 Common 보유 → 해금
```
